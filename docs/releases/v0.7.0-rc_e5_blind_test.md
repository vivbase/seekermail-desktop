# E5 Style-Learning Blind Test — v0.7.0-rc (T107 §3a / §6)

Record for the E5 confusability check. Pass criterion: **confusability ≥ 40%**
(the rater's error rate when guessing AI vs. human), per AI_MODES_DESIGN §11 /
planning/05 §5.3.

## Protocol

1. Import 50 of the user's historical sent mails as the style corpus; trigger
   style learning for the account.
2. Generate 5 AI reply drafts for 5 held-out incoming mails.
3. Take the 5 matching real user replies to those same mails.
4. Shuffle all 10 into a random order (record the key separately).
5. The Release Engineer (who did not author the mails) labels each "AI" or
   "Human" without feedback.
6. Confusability = wrong labels ÷ 10. ≥ 0.40 passes.
