## Notes

- **Path abstraction (#1):** `Paths::resolve()` uses `directories::BaseDirs::data_dir()`,
  which returns the correct per-OS root with no hardcoded macOS string. Verified by
  `grep -r "Library/Application Support" src-tauri/src/` → no matches.
- **Fonts (#5/#6):** the `html.script-*` font stacks all end in a system `sans-serif`
  fallback; Windows ships Segoe UI (Arabic) and Microsoft YaHei / Yu Gothic (CJK), so
  the fallback chain resolves without tofu even when Noto fonts are absent.
- Windows ships as **Public Beta** at v1.0 (`planning/05 §5.4`); these manual rows
  are confirmed once before the GA tag and again at the first Windows beta build.
